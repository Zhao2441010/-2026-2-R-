4.关于启动
普通程序通过main函数+三角启动    之前的c++,py版本都是这种

spring boot 通过 "启动类" 来启动spring容器

启动类:类似于main:
@SpringBootApplication  // ← 这是一个组合注解等于同时写了三个
public class MyApplication {
    public static void main(String[] args) {
        SpringApplication.run(MyApplication.class, args);  // ← 不是打印Hello，而是"启动Spring"
    }
}

spring容器:
自动找带@Component的类,所以要在文件里加上@Entity等
