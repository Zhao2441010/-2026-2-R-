1.springboot+meavn项目,文件分区放置
demo
    src
        text  测试代码 

        main
            resources   数据库代码
            java
                \com\example\demo
                    
                    \entity      放置java类
                    \service     业务逻辑代码
                    \repository  数据库操作
                    \controller  web接口
                    \util        工具比如多线程


2.数据库

    数据库配置,在src/main/resources/application.yml配置取代conn={}
    
    数据库操作接口写在src\main\java\com\example\demo\repository

        @Repository
        public interface CatRepository extends JpaRepository<Cat, Long> {
        }

    增删改查在src\main\java\com\example\demo\server
    通过调用接口实现

        @Autowired
        private CatRepository catRepository;

        // 查
        public List<Cat> list() {
            return catRepository.findAll();
        }
